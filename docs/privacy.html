Política de Privacidad - Mejengas

Última actualización: 2026

Mejengas recopila información básica cuando el usuario decide iniciar sesión con Google.

Información que recopilamos

  • Nombre
  • Correo electrónico
  • Foto de perfil (si está disponible)
  • ID del dispositivo para notifiaciones PUSH

Uso de la información

La información se utiliza únicamente para crear y gestionar la cuenta del usuario dentro de la aplicación.

Compartición de datos

No vendemos ni compartimos información personal con terceros.

Seguridad

Utilizamos servicios seguros para almacenar la información.
